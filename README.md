# 📋 Attendance Management System

An **Attendance Management System (AMS)** is a web-based application designed to manage and track attendance efficiently.  
This system replaces manual attendance methods with a digital solution, improving accuracy and saving time.

The project is developed using **Angular (Frontend)** and **C# .NET (Backend)**.

---

# 🚀 Features

- 🔐 User Login Authentication  
- 👨‍🎓 Student Management  
- 📅 Mark Attendance  
- 📊 View Attendance Records  
- 🔍 Search Attendance by Date  
- 📈 Dashboard Overview  
- 🧾 Generate Attendance Reports  
- 👨‍💼 Admin Panel
